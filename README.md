# 📖 About

**Chatlet** is an AI-powered conversational web application inspired by Google Gemini.

The application allows users to communicate with Google's Gemini Large Language Model in real time through an elegant chat interface. It generates intelligent responses, remembers the ongoing conversation during the session, and delivers a smooth user experience using React's Context API for state management.

Built using **React**, **Vite**, and the **Google Generative AI SDK**, Chatlet demonstrates the integration of modern frontend technologies with generative AI APIs.

# 🏛️ System Design

The application follows a client-side architecture where the React frontend interacts directly with Google's Gemini API using the official SDK.

```text
                     User
                       │
                       ▼
             React + Vite Frontend
                       │
                       ▼
              Context API State
                       │
                       ▼
          Google Generative AI SDK
                       │
                       ▼
                Gemini AI Model
                       │
                       ▼
              AI Generated Response
                       │
                       ▼
               Render Conversation
```

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/asyncShaurya/chatlet.git
```

Move into the project

```bash
cd chatlet
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

# ⚙️ Environment Variables

Create a `.env` file in the root directory.

```env
VITE_GEMINI_API_KEY=YOUR_GOOGLE_GEMINI_API_KEY
```
